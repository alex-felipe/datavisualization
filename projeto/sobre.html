INÍCIO MAIS INFORMAÇÕES SOBRE O PROJETO DATA HACKERS

OBJETIVO DO PROJETO


Este projeto tem por objetivo apresentar informações sobre o mercado de trabalho na área de ciência de dados no Brasil. Essas informações visam contribuir para que estudantes e outros interessados na área possam ter uma visão geral sobre o mercado de trabalho da profissão de Cientista de Dados no Brasil. O projeto tem como base os resultados de um survey realizado pela comunidade Data Hackers publicados em Maio de 2020. O survey realizado pela comunidade coletou dados como salários, formações e tempo de experiência de mais de 1700 cientistas de dados do Brasil. Com isso, este projeto irá estruturar e apresentar visualizações que irão auxiliar na compreensão e interpretação desses dados


PÚBLICO ALVO


O principal público alvo deste projeto são os estudantes da área de TI e outros interessados em Ciência de Dados.


DESCRIÇÃO DOS DADOS:


Os dados que serão utilizados neste projeto são resultados de um survey disponibilizados pela comunidade Data Hackers em Maio de 2020. Esses dados foram disponibilizados em formato CSV juntamente com o dicionário das perguntas realizadas no survey em formato TXT. O arquivo contém 36 colunas referentes às perguntas realizadas no survey, sendo 5 delas mascaradas (entrada de dados subjetivos dentro de um formato especifico) e as outras questões com alternativas. Além disso, foram inseridas 6 colunas derivadas a partir das respostas obtidas nas outras questões. Como tratamento dos dados disponibilizados, será realizada a conversão do arquivo CSV para JSON, visando facilitar o manuseio na plataforma online. Também será necessário o agrupamento das questões com alternativas para uma única coluna onde um identificador irá representar a alternativa selecionada e a correções de erros com a codificação dos caracteres.


PERGUNTAS DE PARTIDA / HIPÓTESES


Como o objetivo deste projeto é apresentar informações sobre o mercado de trabalho para a profissão de Cientista de Dados no Brasil, seguem algumas das principais questões a serem respondidas:

  • Pessoas que começam na área, geralmente ganham quanto? A regra vale para todas as regiões e estados?
  • Como está o gender gap no Brasil? Há diferença salarial entre homens e mulheres na área?
  • Há muitas pessoas de áreas não-exatas trabalhando com Data Science?
  • Cargos de gestão ganham mais do que cargos técnicos?
  • Quais bancos de dados/fontes de dados são mais utilizados pelos Cientistas de Dados participantes da pesquisa?
  • Quais as principais ferramentas utilizadas pelos Cientistas de Dados participantes da pesquisa?
